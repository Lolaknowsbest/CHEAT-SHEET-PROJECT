CSS SELECTORS CHEAT SHEET

Type of selectors mostly used in CSS and how to use them.

Selector Name HTML Description CSS Description
Universal Selector This "*" selector selects all elements * { }
Class Selector The .class selector selects elements with a specific class attribute e.g.
<.. a class="x"> 		.a { }
Tag Selector a a { }
ID Selector <.. id="a"/> #a
Child Selector The child selector selects all elements that are direct children of a specified element.
This e.g selects all "b" elements that are children of "a" element		 a > b
Descendent Selector The descendant selector matches all elements that are descendants of a specified element.
The following example selects all "c" elements inside "a" elements		 a c { }
Adjacent Sibling Selector Adjacent Sibling Selector (+)
The adjacent sibling selector is used to select an element that is directly after another specific element.
Sibling elements must have the same parent element, and "adjacent" means "immediately following".
The following example selects the first "d" element that are placed immediately after "a" elements		 a + d { }
General Sibling Combinator The general sibling selector selects all elements that are siblings of a specified element
Must be children of the same parent element. 		a ~ b

These are just a few examples of common selctors used to style and format elements in css
